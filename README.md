📊 Sales Data Analysis Project
🔍 Objective

Analyze sales dataset to understand business performance, profitability, and key trends.

🛠 Tools Used
Python
Pandas
Matplotlib
Jupyter Notebook

📈 Key Insights
Total Sales: ~2.29 Million
Total Profit: ~286K (Profit Ratio: ~12.4%)
West region is the most profitable
Central region is least profitable due to losses in Furniture (especially Furnishings)
Sales peak at end of year and drop at beginning
Technology category generates highest sales# Sales-Data-Analysis
